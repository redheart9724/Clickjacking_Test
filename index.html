📋 Response header inspector

      
Checks for X-Frame-Options, Content-Security-Policy: frame-ancestors, and Permissions-Policy. Most sites don't send CORS headers that let a browser page read these cross-origin, so this falls back to a ready-to-run terminal command when the browser can't see them.

⏱️ Timing / multi-step bypass test

      
Some confirmation flows only need two clicks in quick succession — a common bypass for defenses that expect a single deliberate click. This times a two-click sequence so you can note the window a real overlay would need.
